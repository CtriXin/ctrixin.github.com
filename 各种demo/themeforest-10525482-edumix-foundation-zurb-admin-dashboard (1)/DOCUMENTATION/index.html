theme.css & style.cssfile imports all alternative stylesheets,This is the main css styles for this template. Please keep in mind also that all the entries in the css is not ONLY in the folder css/.., but also in the js/.. it is mainly for css required by plugin of certain jquery

We also have to include SASS file, which you can see in the sass folder. Sass lets you use features that don't exist in CSS yet like variables, nesting, mixins, inheritance and other nifty goodies that make writing CSS fun again.

Here's the structure:

  • contetnt.scss —Main scss
  • sidebar.scss —for style of sidebar menu
  • style.scss —for main style of content menu
  • topbar.scss —for style oftop menu
  • theme.scss —for main theme style
  • dashboard.scss —for style of dashboard
